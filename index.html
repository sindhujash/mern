Mangoodb

MongoDB is a document-oriented, open-source, nonrelational database management system (DBMS) that stores data in flexible, JSON-like documents

Express js

Express.js is a minimalist framework built on top of Node.js that simplifies the process of creating server-side applications. It provides tools for routing, middleware, and handling HTTP requests and responses. Express is commonly used to build REST APIs and web servers, making it a popular choice for backend development.

React js

React.js is a JavaScript library for building dynamic user interfaces, particularly single-page applications (SPAs). It is based on a component-driven architecture, allowing developers to create reusable UI components. React uses a virtual DOM to optimize rendering and improve performance, making it a favorite for modern web development.

Node js

Node.js is a JavaScript runtime environment that allows developers to run JavaScript code on the server side. It is known for its event-driven, non-blocking architecture, making it ideal for building scalable and high-performance applications such as real-time chat apps or APIs. Node.js uses npm (Node Package Manager) for managing libraries and dependencies.
